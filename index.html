카카오 API 키 설정

  1. developers.kakao.com 접속 → 로그인
  2. 내 애플리케이션 → 애플리케이션 추가
  3. 앱 설정 → 플랫폼 → Web → 사이트 도메인 추가
    http://localhost 또는 파일 경로
  4. 앱 키 → JavaScript 키 복사
  5. index.html 3번째 script 태그의
    YOUR_KAKAO_API_KEY 를 키로 교체

⚠️ REST API 키가 아닌 JavaScript 키를 사용하세요.
